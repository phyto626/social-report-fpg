# 洗衣精補充站社群成果分析與簡報製作技能

此技能旨在自動化「洗衣精補充站」的社群內容分析與成果簡報製作流程。它將根據提供的社群數據，深入分析貼文表現，歸納受歡迎內容特徵，提出優化建議，並規劃未來活動策略，最終生成一份符合品牌視覺規範的精簡版 HTML 報告。

## 啟動前必做：詢問粉專追蹤人數

**每次執行此技能前，必須先詢問使用者當期的粉專追蹤人數。**
不可使用任何預設值或上次的數字。確認數字後才能開始分析與產出報告。

範例提問：「請問粉專目前的追蹤人數是多少？」

## 功能說明

1. **貼文數據分析**：
   - 貼文發布篇數：自動從 Excel 總列數計算。
   - 粉專追蹤人數：每次向使用者確認，填入 KPI 區塊。
   - 總觸及人數、平均互動次數：自動加總／平均計算。
   - **貼文互動率算法**：(心情數 + 留言數 + 分享次數) / 觸及人數。
2. **受歡迎內容特徵歸納**：識別高表現貼文的共同特徵。
3. **優化方向建議**：提供基於數據分析的社群內容優化策略。
4. **未來活動策略規劃**：提出具體的社群活動與合作建議。
5. **自動生成 HTML 報告**：輸出至 `/mnt/user-data/outputs/`，並以 `present_files` 工具提供下載。

## 視覺化規範

為確保品牌一致性與專業度，所有生成的視覺化內容將嚴格遵循以下規範：

- **背景**：簡約白色背景（`#f7f9fc`）。
- **字體**：必須使用 **Noto Sans TC**（Google Fonts）確保中文顯示正常。
- **品牌配色方案**：
  - 主色：`#0057B7`
  - CTA色：`#FFA000`、`#00A0BB`、`#5CBEB2`
  - 輔助色：`#D4EAED`、`#EFEFED`、`#FFFFFF`

## 輸入格式

Excel 欄位對應：

| 欄位名稱 | 說明 |
| :--- | :--- |
| `標題內容` | 貼文主題分類 |
| `素材型式` | 照片／圖片／短影音 |
| `貼文內容` | 社群貼文文字 |
| `觸及人數` | 貼文觸及人數 |
| `心情、留言和分享次數` | 三項互動合計數（單一欄位） |

## 報告結構與規格

輸出為單一 HTML 檔案，包含以下區塊（順序固定）：

### 1. 頁首（Report Header）
- 漸層背景：`#003d82` → `#0057B7` → `#00A0BB`
- 顯示：品牌名稱、報告副標、分析篇數、報告年份

### 2. KPI 概覽（4 格）
固定順序與內容：
| 位置 | 指標 | 顏色 |
| :--- | :--- | :--- |
| 第 1 格 | 貼文發布總篇數 | 藍色 `#0057B7` |
| 第 2 格 | 粉專追蹤人數（每次向使用者確認） | 橘色 `#FFA000` |
| 第 3 格 | 累計總觸及人數 | 青色 `#00A0BB` |
| 第 4 格 | 平均互動次數（心情＋留言＋分享） | 綠色 `#5CBEB2` |

### 3. 本期互動率最高貼文 Highlight
- 自動找出互動率最高的單篇貼文
- 顯示：主題、貼文摘要（前 80 字）、觸及、互動次數、互動率、素材型式

### 4. 內容表現分析（左右兩欄）
- **左欄**：各主題互動率排名表（含互動率進度條，最高值 = 100%）
- **右欄**：
  - 素材型式分析（橫向進度條 + 平均觸及、互動率）
  - 數據洞察文字說明
  - 發文數量長條圖

### 5. 全貼文表現明細表
- **欄位**：序號、貼文主題、素材、觸及人數、互動次數、互動率（進度條）
- **排序**：依觸及人數由高到低排列
- 互動率進度條以本期最高互動率貼文 = 100% 為基準

### 6. 受歡迎內容特徵歸納（2×2 格 Insight Cards）
- 每格含：emoji 圖示、標題、說明文字
- 邊框顏色依序：橘、青、藍、綠

### 7. 優化方向建議（3 欄 Strategy Cards）
- 每欄含：編號、標題、條列建議（4 點）
- 頂部漸層依序：藍→青、橘→深橘、綠→青

### 8. 未來活動策略規劃（表格）
- 欄位：時機、活動方向、素材建議、預期效益
- 至少 4～5 列，依數據洞察推導具體建議

### 9. 頁尾
- 品牌名稱 + 總觸及 + 分析篇數

## 輸出規格

- 格式：單一 `.html` 檔案
- 儲存路徑：`/mnt/user-data/outputs/`
- 檔名建議：`{品牌}社群成果分析報告.html`
- 完成後以 `present_files` 工具提供給使用者下載
