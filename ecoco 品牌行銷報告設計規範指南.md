# ecoco 品牌行銷報告設計規範指南

為確保 ecoco 品牌行銷報告在視覺呈現上的一致性與專業度，本指南根據 ecoco 企業識別系統 (CIS) 手冊 [1] 整理出以下設計規範。遵循這些準則將有助於強化品牌形象，並有效傳達品牌訊息。

## 1. 背景 (Background)

行銷報告的背景應以簡潔、清晰為主，以確保內容的易讀性。建議採用品牌延伸色中的 **白色** (HEX: #ffffff) 或 **淺膚色** (Pantone P 17-2 C, HEX: #FAE0B8) 作為主要背景色。這有助於突出報告內容，並維持整體視覺的明亮與專業感。

## 2. 字體 (Typography)

字體選擇對於報告的閱讀體驗至關重要。請根據內容語言，嚴格遵循以下字體規範：

| 語言 | 字體名稱 | 應用範疇 |
| :--- | :------- | :------- |
| **英文** | Helvetica Neue, Futura Blood | 標題、內文、強調文字 |
| **中文** | Noto Sans CJK TC, Noto Sans CJK TC Blood | 標題、內文、強調文字 |

**Noto Sans CJK TC Blood** 應用於中文的粗體呈現，以確保視覺上的強調效果與整體風格的一致性。

## 3. 主色 (Main Color)

ecoco 品牌的主色是其視覺識別的核心，應在行銷報告中作為主要視覺元素使用，例如標題、圖表、重點區塊等。

| 色彩名稱 | Pantone 色號 | HEX 值 | RGB 值 | CMYK 值 |
| :------- | :----------- | :----- | :----- | :------ |
| **橘色** | Pantone Orange 021 C | #FF5000 | 255, 80, 0 | 0, 81, 100, 0 |
| **藍色** | Pantone Blue 072 C | #060E9F | 6, 14, 159 | 100, 96, 16, 0 |
| **黃色** | Pantone 116 C | #FFCE00 | 255, 206, 0 | 2, 25, 92, 0 |

## 4. CTA 色 (Call-to-Action Color)

Call-to-Action (CTA) 元素旨在引導讀者採取特定行動，因此其色彩應具有高度的視覺吸引力與對比度。建議使用 ecoco 主要品牌色中的 **橘色** (Pantone Orange 021 C, HEX: #FF5000) 作為 CTA 元素的主要色彩。此色彩鮮明且充滿活力，能有效吸引讀者目光，促使其進行互動。

## 5. 輔助色 (Auxiliary Color)

輔助色用於補充主色，為報告增添視覺層次感，可用於次要資訊、背景區塊或圖表中的輔助線條等。

| 色彩名稱 | Pantone 色號 | HEX 值 | RGB 值 | CMYK 值 |
| :------- | :----------- | :----- | :----- | :------ |
| **白色** | N/A | #ffffff | 255, 255, 255 | 0, 0, 0, 0 |
| **淺膚色** | Pantone P 17-2 C | #FAE0B8 | 250, 224, 184 | 0, 15, 31, 0 |
| **深藍色** | Pantone 7690 C | #0076A9 | 0, 118, 169 | 88, 49, 22, 0 |
| **淺藍色** | Pantone 550 C | #8EB8C9 | 142, 184, 201 | 49, 20, 19, 0 |

## 6. 其他待補充事項

為確保行銷報告的整體視覺品質與品牌一致性，請注意以下事項：

*   **標誌使用**：在報告中置入 ecoco 品牌標誌時，請務必遵循其比例、安全區域及組合形式規範。避免任意變形、扭曲或更改顏色 [1]。
*   **輔助圖案**：可適度運用品牌輔助圖案（如「e」、「c」、「o」字母元素組成的重複圖案或點狀矩陣圖案）作為報告的裝飾元素，以豐富視覺效果，但應避免過度使用導致視覺混亂 [1]。
*   **圖片與圖表**：所有圖片與圖表應保持清晰、專業，並與品牌的整體視覺風格協調一致。建議使用品牌色系進行圖表設計。
*   **排版與間距**：保持頁面排版的整潔與一致性，適當的留白與間距有助於提升閱讀舒適度。
*   **避免使用禁忌**：嚴禁隨意更改品牌標誌的比例或顏色，避免在複雜背景下使用未經處理的標誌，以及添加未經授權的視覺元素 [1]。
